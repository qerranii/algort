В одном далёком королевстве был создан таинственный код, который скрывал важную информацию для правителей. Этот код был записан в виде строк на свитке, и лишь тот, кто разгадает его, сможет открыть секреты. Каждая строка представляла собой зашифрованное сообщение, а для того, чтобы понять, что в нём скрыто, нужно было правильно обработать несколько запросов.

Вам даются несколько строк. Вам нужно выполнить несколько типов запросов, чтобы извлечь или объединить части строк. От того, насколько быстро и точно вы выполните задания, зависит, откроется ли тайна кода.

Входные данные:
Число 
N — количество строк в массиве (1 ≤ N ≤ 100).
N строк, каждая из которых состоит из маленьких латинских букв (длина каждой строки от 1 до 100).
Число 
Q — количество запросов (1 ≤ Q ≤ 100).
Q запросов. Каждый запрос представлен следующим образом:
Если запрос типа 1: целое число 
K (1 ≤ K ≤ 100) — количество символов, которые нужно извлечь из каждой строки.
Если запрос типа 2: два целых числа 
L и R (1 ≤ L ≤ R ≤ 100) — границы подстроки, которую нужно извлечь из каждой строки.
Выходные данные:
Для каждого запроса выведите строку, полученную в результате обработки всех строк.
